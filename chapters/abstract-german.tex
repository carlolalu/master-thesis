Das Ziel dieser Masterarbeit ist die Herleitung von effektiven Gleichungen für ein mikroskopisches Modell (ein System von stochastischen Differentialgleichungen) für die \emph{Chemotaxis} von $N$ Amöben, die vom Molekül von cAMP (zyklisches Adenosinmonophosphat), das sie selbst ausscheiden, mit einem gaussartigen Potential angezogen werden.
Diese Partikel von Chemotaxin werden stochastisch wellenartig abgegeben, die zeitlich einer Poissonverteilung folgen.
Das mikroskopische Modell ist von \cite{ana} inspiriert, das auch die Quelle der vielen Techniken ist (zusammen mit \cite{peter-boers}), die hier benutzt wurden, um das Endresultat zu beweisen.
Darüber hinaus erörtern wir hier die Ähnlichkeiten und die Unterschiede dieses Modells zu dem klassichen makroskopischen Modell für Chemotaxis, nämlich die Gleichung von \emph{Keller-Segel}.
