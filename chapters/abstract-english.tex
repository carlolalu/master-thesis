The aim of this master's thesis is to derive the effective equations for a microscopic system for \emph{chemotaxis} of $N$ amoebas attracted to the cAMP molecules that they released in randomly distributed pulses.
The attractions is given by a Gaussian-like potential, and the times for the creation of particles are Poisson distributed in time.
This microscopic model is inspired from \cite{ana}, which is also the main source of techniques that we adapted for the proof, together with \cite{peter-boers}.
Furthermore, we discuss the similarities and differences of the macroscopic description found with the classic equation for chemotaxis, namely \emph{Keller-Segel}.
