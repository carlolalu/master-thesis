\section{The SDEs framework and the conditions for existence and uniqueness}

An SDE is an expression of the form:

\begin{equation}\label{ch3:eq:general-sde}
\left\{
\begin{aligned}
d X_t &= \drift(t, X_t) dt + \diffusion(t, X_t) d B_t	\\
X_0 &\sim \rho_0
\end{aligned}
\right.
\end{equation}

which is a more conceptual notation to formally mean

\begin{equation}\label{ch3:eq:general-sde-integral-form}
X_t = \rho_0 + \int_0^t \drift(t, X_t) \,dt + \int_0^t \diffusion(t, X_t) \,d B_t
\end{equation}

where $\drift \in \R^\Iu$, $\diffusion \in \R^{\Iu \times \Ch}$ are given vectorial and matricial functions respectively (the given initial distribution $\rho$ is optional).
They are called \emph{drift} and \emph{volatiliyt} respectively.
We sign such SDE as a $E(\drift,\diffusion)$.
\begin{definition}
A solution for $E(\drift,\diffusion)$ consists of
\begin{itemize}
\item a complete filtered space $(\Omega, \Flt, (\Flt_t)_t, \Prb) $
\item an $\Ch$-dimensional $(\Flt_t)_t$-Brownian motion $B = (B^1, \ldots, B^\Ch)$
\item an $(\Flt_t)_t$-adapted $\Iu$-dimensional process $X_t$ with $\Cspc^0$ sample paths
\end{itemize}
such that they satisfy the equation $E(\drift,\diffusion)$.
If furthermore $X_0 = x$ we say that $X$ solves $E_x(\drift,\diffusion)$.
\end{definition}

To ensure existence (strong existence, meaning that we can find a solution on each probability space) and uniqueness (path uniqueness, meaning that having two solutions with the same initial conditions, they are indistinguishable)of solutions we need to require conditions on $\drift, \diffusion$. There are two main results:
\begin{itemize}
\item \autocite[Thm 8.3]{le-gall}: The two coefficients are Lipschitz (eventually only locally) in space and continuous:
\begin{equation*}
\left\{
\begin{aligned}
\forall t .( |\drift(t, x)-\drift(t,y)| &< L_\drift |x-y| &\text{ and } |\diffusion(t, x)-\diffusion(t,y)| &< L_\diffusion |x-y|) \\
\drift &\in \Cspc^0 &\text{ and }\diffusion &\in \Cspc^0
\end{aligned}
\right.
\end{equation*}
\item \autocite[Thm 9.4]{baldi}: The two coefficients are Lipschitz in space (as before) and have sublinear growth in space:
\begin{equation*}
\left\{
\begin{aligned}
\forall t .(|\drift(t, x)-\drift(t,y)| &< L_\drift |x-y| &\text{ and } |\diffusion(t, x)-\diffusion(t,y)| &< L_\diffusion |x-y|) \\
\forall t |\drift(t, x)| &< M_b(1 + |x|) &\text{ and } |\diffusion(t, x)| &< M_\diffusion (1 + |x|)
\end{aligned}
\right.
\end{equation*}\label{ch3:cond:E-and-U-lipsch+subgrowth}
\end{itemize}

\section{Reducing our system to the SDE framework}

\subsection{Defining what is a solution in terms of SDEs}

Our system \ref{ch2:model:deterministic-parenting} is not a proper SDE because of technical oddities:
\begin{itemize}
\item An hypothetical vector $\modsol_t := (\amba_t^1,\ldots, \amba_t^N, \camp_t^{p,g} \text{ already spawned at time }t)$ would have a dimension changing (growing, in particular) as the time flows.
And it grows up to infinity.
\item Not only it would be changing with the time, but it would be a \emph{stochastic} dimension.
\item If we were to write the equations for $\modsol$ in a similar way to the one in $\ref{ch3:eq:general-sde}$ we would have a drift term $\drift = \drift(t, \modsol_t, (\hmactive_t^p)_p)$ depending also on the Poisson-distributed processes $(\hmactive_t^p)_p$, making it "too much stochastical" to be treated as in \ref{ch3:eq:general-sde}.
\end{itemize}

All these oddities originate from the fact that our system depends on $N$ sequences $(\btime^{p,g})_{g}$ of \emph{random} birth times (or equivalently from the Poisson processes generating them, \ie $(\hmactive_t^p)_t$).
We can notice that, while the movement of amoebas $\amba^p$ and cAMP molecules $\camp^{p,g}$ depends on them, such sequences do not depend on $\amba^p, \camp^{p,g}$.
Then heuristically we can imagine to fix these sequences to an arbitrary value $(\tau^{p,g})_{p,g} = (\btime^{p,g})_{p,g}$ (obtaining an equivalent $(\mu_t^p)_{p,t}=(\hmactive_t^p)_{t,p}$), and inquire if the yielded system \ref{ch3:heuristic-fix-the-times} has a solution for it:

\begin{equation}
\left\{
\begin{aligned}
\amba_t^p 
&= \amba_0^p 
+ \int_0^t 
	\overbrace{
		\frac{1}{N} \sum _{p = 1}^{N} \sum_{g = 1}^{\mu_s^p} \attraction(\amba_s^p - \camp_s^{p,g})
	}^{=:\drift_p(s, \modsol_s, (\mu_s^p)_p)=:\drift_p(s, \modsol_s)} 
d s
+ 
\overbrace{\sigma^A}^{=:\diffusion_p(s,\modsol_s)} B_t^p
&
\\
\camp_t^{p,g} 
&\qquad\text{does not exist} 
&if \quad t < \tau^{p,g}
\\
\camp_t^{p,g}
&= \amba_{\tau^{p,g}}^p + \underbrace{\sigma_C}_{=:\diffusion_{p,g}(s,\modsol_s)} \quasibm_t^{p,g}
&if \quad t \geq \tau^{p,g}
\end{aligned}
\right.
\end{equation}\label{ch3:heuristic-fix-the-times}

This solves the problem because $(\tau^{p,g})_{p,g}$ is deterministic.
Since the value is arbitrary, if we find a solution $(\modsol, B) ((\tau^{p,g})_{p,g})$ for \ref{ch3:heuristic-fix-the-times}, then we can assemble all these components into a global solution $(\modsol, B)$.
This is formally explained by defining defining a solution for the system \ref{ch2:model:deterministic-parenting} as a triple $((\hmactive^p)_p, \modsol = (\amba, \camp), B)$ such that under the conditional probability measure

\begin{equation}
\overline{\Prb} : = \PrbCondTimes{\cdot} = \quad\Prb(\ \cdot \ | (\hmactive^p)_{1\leq p\leq N, t\in\R_+} )
\end{equation}

the couple $(\modsol,B)$ solves \ref{ch2:model:deterministic-parenting}.
But we can notice that even in this way the system has a dimension that changes in time, meaning that we need to operate piecewise in time (\ie define what is a solution that solves \ref{ch2:model:deterministic-parenting} as a solution that solves it piecewise in time).
We can consider the birth times chronologically as in the \ref{ch2:model:casual-parenting}, \ie

\begin{equation}
\btime_0:=0 < \btime_1 < \btime_2 < \ldots < \btime_m < \ldots
\end{equation}

and define the correspondent intervals

\begin{equation}
I_0 := [0,\btime_1], I_1:=[\btime_1,\btime_2], \ldots , I_m:= [\btime_1,\btime_{m+1}],\ldots
\end{equation}

On each interval $I_m$ the system can be described as this:

\begin{equation}
\left\{
\begin{aligned}
t &\in I_m = [\btime_m,\btime_{m+1}]
&
\\
\amba_t^p 
&= \amba_{\btime_m}^p 
+ \int_{\btime_m}^t 
	\frac{1}{N} \sum _{p = 1}^{N} \sum_{g = 1}^{\hmactive_s^p} \attraction(\amba_s^p - \camp_s^{p,g})
d s + \sigma^A B_t^p
&
\\
\camp_t^{p,g} 
&= \camp_{\btime^{p,g}}^{p,g} + \sigma_C \quasibm_t^{p,g}
&\text{if} \quad \btime^{p,g} < \btime_m
\\
\camp_t^{p,g}
&= \amba_{\btime^{p,g}}^p + \sigma_C \quasibm_t^{p,g}
&\text{when} \quad \btime^{p,g} = \btime_m
\end{aligned}
\right.
\end{equation}\label{ch3:det-parent-piece}

\begin{remark}
We need to observe that only in $I_m \smallsetminus \{\btime_{m+1}\}$ the number of particles (and thus the dimension) is constant, so why can we claim that \ref{ch2:model:deterministic-parenting} is described efficiently by \ref{ch3:det-parent-piece} efficiently on $I_m$?
\begin{itemize}
\item Since the $\{\btime_{m+1}\}$ has Lebesgue-measure $0$ the trajectory of the particles already spawned in $I_m \smallsetminus \{\btime_{m+1}\}$ is not influenced in $I_m$ by $\camp^{m+1}$.
Then we can ignore its existence also in $I_m$ and have thus a constant dimension on such interval.
\item The particle $\camp^{m+1}$ spawns on the point $\amba_{\btime_{m+1}}^p$ for some $p$, thus the solution on the interval $I_{m+1}$ can be glued with "continuouty" with the solution on $I_{m}$ even regarding the spawning position of $\camp^{m+1}$, meaning that if we would define the functions
\begin{equation}
\begin{aligned}
\Zhi_m: I_m &\longrightarrow \R_+
\\
		t &\longmapsto	max \{|\modsol^i - \modsol^j| \}
\end{aligned}
\end{equation}
then we could define a globally continous function $\Zhi:\R \longrightarrow \R_+$ piecewise defined as $\Zhi_m$.
\end{itemize}
\end{remark}

\begin{remark}
In the interval $I_m$ the number of particles is
\begin{equation}
\NumParticles_t := N + \sum_{p=1}^N \hmactive_t^p = \overbrace{N}^{\# \text{amoebas}} + \overbrace{\hmactive_t}^{\# \text{cAMP present at time } t }
\end{equation}
thus the dimension of the system is
\begin{equation}
\NumParticles_t \cdot d
\end{equation}
where $d$ is the dimension in which we are considering the particles ($2$ or $3$).
\end{remark}

The only passage that we need in order to describe \ref{ch3:det-parent-piece} as a proper SDE is that of transforming the quasi-Brownian motions $\quasibm^{p,g}$ into proper Brownian motions.
This is easily done by shifting the system in time from $I_m$ to $I_m-\btime_m=[0, \btime_{m+1}-\btime_m]$, obtaining:

\begin{equation}
\left\{
\begin{aligned}
t &\in I_m - \btime_m = [0, \btime_{m+1}-\btime_m]
&
\\
\amba_t^p 
&= \amba_{\btime_m}^p
	+ \int_0^t 
		\overbrace{
			\frac{1}{N} \sum _{p = 1}^{N} \sum_{g = 1}^{\hmactive_s^p} \attraction(\amba_s^p - \camp_s^{p,g})
		}^{\drift_p}
	d s
	+ 
	\overbrace{\sigma^A}^{\diffusion_p} 
	\overbrace{(B_{t + \btime_m}^p - B_{\btime_m}^p)}
		^{\text{Brownian motion in }t}
&
\\
\camp_t^{p,g} 
&= \camp_{\btime^{p,g}}^{p,g} + \sigma_C \quasibm_{\btime_m + t}^{p,g} = \camp_{\btime^{p,g}}^{p,g} + \sigma_C
	\overbrace{(B_{t + \btime_m}^{p,g} - B_{\btime_m}^{p,g})}
	^{\text{Brownian motion in }t}
&\text{if} \quad \btime^{p,g} < \btime_m
\\
\camp_t^{p,g}
&= \amba_{\btime^{p,g}}^p + \sigma_C \quasibm_{\btime_m + t}^{p,g}
	= \amba_{\btime^{p,g}}^p + \underbrace{\sigma_C}_{\diffusion_{p,g}} B_t^{p,g}
&\text{when} \quad \btime^{p,g} = \btime_m
\end{aligned}
\right.
\end{equation}\label{ch3:final-SDE-reduction-determ-parenting}

and if this system has a solution ${}^m Y_t$ on $I_m - \btime_m$, then it is straightforward to verify that ${}^m \modsol_t := {}^m Y_{t-\btime_m}$ solves \ref{ch3:det-parent-piece} on $I_m$.
To be accurate we should operate also on the Brownian motion: if $B^{*}_t$ fullfills \ref{ch3:final-SDE-reduction-determ-parenting} for all particles but the one which just spawned, then $B^{*}_{t+\btime_m}$ fullfills \ref{ch3:det-parent-piece} for the equivalent equations; while if $B^{p,g}_t$ is the B.m. of the cAMP spawning in \ref{ch3:final-SDE-reduction-determ-parenting}, then the correpsondent quasi-B.m. $\quasibm_t^{p,g}$ (defined as in \ref{ch2:quasi-bm-camp}) fullfills the equation in \ref{ch3:det-parent-piece} for the correspondent particle.

Then we can say that $(\modsol, B)$ is a solution for the piecewise defined system \ref{ch2:model:deterministic-parenting} under $\PrbCondTimes{\cdot}$ if it solves (under the same random measure) the system \ref{ch3:final-SDE-reduction-determ-parenting}.

\section{Ensure the existence of solutions?}

Now that we have reduced our model to a piecewise defined SDE, we can find individual solutions to each of them (each is defined on its interval $I_m$) and glue all of them together.
The glued solution will have, by definition, continuous sample paths for each particle from the moment of its existence on.
Even more, since each spawned particle $C^{p,g}$ is born on the position of its parent amoeba $A^p$, we will have a bifurcation there (\ie there are no jumps in the distance between particles at the moment of spawning).
